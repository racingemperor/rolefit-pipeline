# School Signals

This directory defines how local subagents should research school-company cooperation and grade-specific internship/full-time opportunities.

The repository does not contain assumed school-company partnerships. Runtime agents must use official school career centers, college announcements, employer campus pages, school-enterprise cooperation notices, internship-base announcements, and employment reports before claiming a school-specific advantage.

Social media and candidate posts can explain preparation risks or common experiences, but they cannot prove a partnership or guaranteed hiring preference by themselves.
